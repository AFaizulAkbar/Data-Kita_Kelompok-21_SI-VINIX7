🗂️ DataKita — Pendataan Warga Digital

DataKita adalah website berbasis web yang dirancang untuk membantu RT/RW atau perangkat desa dalam mengelola pendataan warga secara digital, rapi, dan efisien tanpa pencatatan manual di buku.

Project ini dibuat sebagai tugas Studi Independen di PT VINIX7 AURUM dan saat ini masih berfokus pada pengembangan landing page (frontend).

⚠️ Catatan: Saat ini project masih berupa tampilan antarmuka (UI) dan belum terintegrasi dengan backend maupun database.

🚀 Cara Menjalankan Project
1️⃣ Clone Repository
git clone https://github.com/username/datakita.git
2️⃣ Masuk ke Folder Project
cd datakita
3️⃣ Buka Project

Menggunakan Visual Studio Code (disarankan):

code .

Atau buka manual melalui VS Code:
File → Open Folder → pilih folder project.

🌐 Menjalankan di Browser

Cukup double-click file index.html, maka website akan langsung terbuka di browser.

⚡ Menggunakan Live Server (Disarankan saat Development)
Buka VS Code
Masuk ke Extensions (Ctrl + Shift + X)
Cari Live Server → Install
Klik kanan index.html
Pilih Open with Live Server

Browser akan otomatis refresh saat ada perubahan kode.
